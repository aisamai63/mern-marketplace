# MERN Marketplace Frontend (Vite + React)

This project is now powered by [Vite](https://vitejs.dev/) and React for a fast, modern development experience.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

Runs the app in development mode.\
Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.

### `npm run build`

Builds the app for production to the `dist` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run preview`

Previews the production build locally at [http://localhost:4173](http://localhost:4173).
